TO DEPLOY ON YOUR IIS SERVER:
==================================================

1. ON YOUR COMPUTER (if you have .NET SDK):
   Run: build.bat
   This creates a 'publish' folder

   OR manually:
   dotnet publish -c Release -o publish

2. COPY TO IIS SERVER:
   Copy all files from 'publish' folder to:
   C:\inetpub\wwwroot\solwebhook
   (or wherever you want)

3. ON IIS SERVER - Install .NET:
   Download and install:
   https://dotnet.microsoft.com/download/dotnet/6.0
   Choose: "Hosting Bundle" (includes everything)

4. ON IIS SERVER - Configure IIS:
   a) Open IIS Manager
   b) Create new Application Pool:
      - Name: SolWebhookPool
      - .NET CLR Version: No Managed Code
   c) Create new Site or Application:
      - Physical path: C:\inetpub\wwwroot\solwebhook
      - Application Pool: SolWebhookPool
      - Binding: Port 80 (or whatever you want)

5. TEST:
   http://your-iis-server/health
   Should return: {"status":"healthy",...}

6. UPDATE QUICKNODE:
   Change webhook URL to: http://your-iis-server/
